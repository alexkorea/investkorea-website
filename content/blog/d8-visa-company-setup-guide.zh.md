## D-8签证与法人设立的关系

D-8（企业投资）签证是外国人在韩国经营事业所需的代表性居留资格。根据《外国人投资促进法》，该签证颁发给在韩国投资**1亿韩元以上**并实际参与法人经营管理的外国人。获得D-8签证后，不仅可以在韩国合法经营事业，还可以通过签证续签实现长期居留，满足一定条件后还可以变更为[F-5永久居留权](/visa/f5)。

D-8签证与法人设立**密不可分**。申请D-8签证前，必须先在韩国设立[外商投资法人（FDI）](/company/fdi)，并准备好投资资金汇款证明和营业执照。也就是说，法人设立是签证申请的前提条件。部分投资者认为可以"先拿到签证再设立法人"，但D-8签证必须在法人设立和投资资金汇款完成后才能申请。

从法人设立到D-8签证发放的全过程涉及多个政府机构（银行、法院、税务局、KOTRA、出入境管理局等），可能较为复杂。但只要准确理解每个步骤并系统地做好准备，整个过程可以在**约3至6周**内完成。本文将详细介绍各阶段的具体流程和注意事项。

## 法人类型比较

外国人在韩国设立企业主要有三种方式。各类型的法律地位、要件及D-8签证关联可能性各不相同，因此根据事业目的和规模选择合适的类型非常重要。

| 类别 | [外商投资法人（FDI）](/company/fdi) | [分公司（Branch）](/company/branch) | [联络处（Liaison）](/company/liaison) |
|------|------|------|------|
| **法律性质** | 独立法人（韩国法人） | 外国总公司的国内分支机构 | 外国总公司的联络据点 |
| **最低投资额** | 1亿韩元以上 | 无（使用总公司资金） | 无 |
| **D-8签证** | 可申请 | 可申请（D-7/D-8） | 不可（可申请D-7） |
| **营业活动** | 全面允许 | 允许（以总公司名义） | 不允许（仅限非营利活动） |
| **税务** | 韩国法人税（全球所得） | 韩国来源所得征收法人税 | 无所得，不适用 |
| **设立难度** | 中等 | 较高（部分行业需许可） | 较低 |
| **适用情况** | 全面进入韩国市场、独立运营 | 总公司业务拓展至韩国 | 市场调查、联络目的 |

**外商投资法人（FDI）** 是获取D-8签证最常见的方式。作为依据韩国法律设立的独立法人，可在韩国自由开展营业活动，税务方面按照韩国法人税法纳税。大多数外国投资者都选择此方式。

**[分公司（Branch）](/company/branch)** 是外国总公司在韩国的分支机构，与总公司具有相同的法人资格。虽然没有单独的投资资金要求，但总公司的财务状况和经营业绩会影响设立审查。金融业、保险业等特定行业常采用分公司形式，可通过[D-7（企业内部调动）签证](/visa/d7)或D-8签证居留。

**[联络处（Liaison Office）](/company/liaison)** 是在韩国不从事营业活动，仅进行市场调查、联络、信息收集等非营利活动的据点。无需单独设立登记，手续简便，但不能从事营业活动，也不符合D-8签证的申请条件。

<div style="background:#EFF6FF; border-left:4px solid #3B82F6; padding:12px 16px; border-radius:0 8px 8px 0; margin:16px 0; font-size:14px;">💡 <strong>提示：</strong>也可以采取分阶段策略——先设立联络处进行市场调研，在正式进入阶段再转为外商投资法人（FDI）。请根据事业规模和战略选择合适的类型。</div>

## 设立要件详解

下面详细了解外商投资法人（FDI）设立的核心要件。任何一项要件未能满足，都可能导致法人设立失败，或者设立后D-8签证被拒，因此必须仔细确认。

| 要件 | 内容 | 详细说明 |
|------|------|-----------|
| **最低投资额** | 1亿韩元以上 | 《外国人投资促进法》规定的最低金额。部分行业可能要求更高金额 |
| **出资比例** | 持有10%以上有表决权的股份 | 外国投资者持股低于10%则不被认定为外国人投资 |
| **营业场所** | 可实际开展业务的地址 | 虚拟办公室、共享办公室均可，但出入境管理局可能进行实地检查 |
| **代表理事** | 外国人或韩国人 | 代表理事为外国人时可申请D-8签证 |
| **经营目的** | 合法营利性事业 | 公司章程中载明的经营目的不得违反韩国法律 |
| **投资者资格** | 外国人或外国法人 | 海外同胞（F-4）、永住权持有者（F-5）等也可进行外国人投资 |

**1亿韩元投资额**以现金投资为基准。也可以通过实物（机器设备、知识产权等）出资，但实物出资需要评估鉴定，流程更加复杂且耗时更长。大多数情况下，现金汇款是最高效的投资方式。

关于营业场所，不一定需要独立办公室，使用**共享办公空间（联合办公）** 的地址也是可以的。但由于出入境管理局可能进行实地检查以确认是否实际开展业务，因此该空间必须是实际进行工作活动的场所。此外，部分行业（制造业、食品业等）可能有额外的设施要求。

代表理事可以是外国人也可以是韩国人。但要申请D-8签证，签证申请人必须**实际参与**法人的经营管理。如果仅持有股份而不参与经营，D-8签证可能会被拒绝。

## 投资资金汇款流程

投资资金汇款是法人设立过程中最重要的环节之一。如果汇款程序不合法，投资将无法被认定为外国人投资，D-8签证也将无法获批。

### 第一步：外国人投资申报

在汇出投资资金之前，必须先向**KOTRA（大韩贸易投资振兴公社）或外汇银行**提交外国人投资申报书。申报书中需填写投资者信息、投资金额、被投资法人信息、经营目的等。申报获批后，将颁发**外国人投资申报确认书**，银行将根据该确认书编号开设投资资金专用账户。

### 第二步：外汇银行开户及汇款

取得外国人投资申报确认书后，在韩国国内**外汇银行**（国民银行、新韩银行、韩亚银行、友利银行等）开设投资资金专用账户。该账户与普通存款账户不同，是专门用于管理外国人投资资金收支的账户。从海外通过**电汇（T/T）** 将投资资金汇入该账户，汇款时须明确标注汇款用途为"外国人投资资金"。

### 第三步：投资资金到账确认书

投资资金到达韩国国内银行账户后，该银行将签发**投资资金到账确认书（外汇购入证明书）**。该文件是D-8签证申请的核心证明材料，证明投资资金是从海外合法汇入的。国内持有的资金或从第三方借入的资金不被认定为外国人投资资金，因此必须确认**从海外到国内**的汇款路径。

<div style="background:#FEF3C7; border-left:4px solid #F59E0B; padding:12px 16px; border-radius:0 8px 8px 0; margin:16px 0; font-size:14px;">⚠️ <strong>注意：</strong>投资资金必须从投资者本人名下的海外账户汇出。从第三方名义账户汇款的，可能不被认定为外国人投资，并可能根据《反洗钱法》接受额外审查。</div>

## 法人设立登记详解

投资资金汇款完成后，向**管辖登记所（法院）** 申请法人设立登记。法人设立登记是官方确认法人法律存在的程序，登记完成后法人才获得法人资格。

### 所需文件

法人设立登记所需的核心文件如下：

| 文件 | 内容 | 备注 |
|------|------|------|
| 公司章程 | 法人目的、名称、总部所在地、资本金、董事构成等基本规则 | 需公证 |
| 股东大会（发起人大会）会议记录 | 法人设立决议、理事选任等记录 | 需公证 |
| 董事会会议记录 | 代表理事选任等记录 | 理事2人以上时需要 |
| 就任承诺书 | 理事、监事的就任同意书 | 附印鉴证明 |
| 印鉴申报书 | 法人印鉴登记 | 法院表格 |
| 股份认购书 | 各股东的股份认购明细 | 按投资者分别准备 |
| 余额证明书 | 投资资金到账确认 | 银行出具 |
| 外国人投资申报确认书副本 | 外国人投资申报证明 | KOTRA/银行出具 |

### 登记流程

公司章程堪称法人的宪法，必须经**公证人公证**。外国投资者还需额外提供投资者本人的护照复印件、本国的印鉴证明（或签名公证）等。所有文件准备齐全后，向法人总部所在地管辖登记所提交设立登记申请书。需缴纳登记许可税和教育税，税额根据资本金规模有所不同。

登记申请后约**3至7天**内完成登记，届时可获得登记完成证明和法人登记簿副本。法人登记簿副本将作为后续营业执照办理、外商投资企业登记、D-8签证申请等所有手续的核心证明文件。

## 营业执照办理流程

法人设立登记完成后，须在管辖**税务局**办理营业执照。营业执照是法人纳税、开具税务发票以及正常开展经营活动的必要手续。

办理营业执照所需的文件包括：营业执照申请表、法人登记簿副本、公司章程副本、租赁合同（营业场所地址证明）、代表理事身份证复印件（外国人提供护照复印件）等。使用共享办公室的，提交与该共享办公室的服务合同。

营业执照通常在申请后**1至3天**内办妥。根据行业类型，如需额外的许可、登记或申报（如食品相关行业、教育相关行业等），须先取得相应许可后再办理营业执照。营业执照签发后，即可正式以法人名义开设银行账户，全面开展经营活动。

## D-8签证申请详解

法人设立、投资资金汇款、营业执照办理全部完成后，最后一步是向**出入境管理局**申请[D-8签证](/visa/d8)。签证申请可在韩国国内出入境管理局（变更居留资格）或海外韩国使领馆（签证签发）办理。

### 所需文件

| 文件 | 内容 | 备注 |
|------|------|------|
| 签证签发（居留资格变更）申请表 | 出入境管理局表格 | 附照片 |
| 护照复印件 | 有效期6个月以上 | 携带原件 |
| 外商投资企业登记证 | 产业通商资源部签发 | 经KOTRA |
| 法人登记簿副本 | 法人设立证明 | 3个月内签发 |
| 营业执照副本 | 税务局签发 | - |
| 投资资金到账确认书 | 海外汇款证明 | 银行签发 |
| 商业计划书 | 经营内容、营收计划、招聘计划等 | 需用韩语撰写 |
| 办公室租赁合同 | 营业场所实存证明 | - |
| 在职证明或经历证明 | 投资者专业性证明 | 如适用 |

### 审查过程

D-8签证审查将综合考查投资资金的合法性、事业的真实性以及投资者参与经营管理的意愿。出入境管理局可能根据需要进行**办公室实地检查**，商业计划书的可行性也是审查要素之一。如果仅以获取签证为目的而形式性地设立法人，签证可能会被拒绝。

签证审查期约为**1至4周**，如需补充材料则需要额外时间。D-8签证的初始居留期限为1至2年，事业正常运营的情况下可以续签。持D-8签证在韩国居留5年以上并满足一定条件（维持投资、雇用韩国国民等），还可以申请变更为[F-5永久居留权](/visa/f5)。

<div style="background:#EFF6FF; border-left:4px solid #3B82F6; padding:12px 16px; border-radius:0 8px 8px 0; margin:16px 0; font-size:14px;">💡 <strong>提示：</strong>D-8签证持有者可以为配偶和未成年子女申请F-3（随行）签证。如计划家人随行，建议在申请D-8签证时一并准备。</div>

## 设立后注意事项

法人设立和D-8签证取得完成后，为确保正常经营运营，仍有一些行政手续需要办理。忽视这些步骤可能会导致罚款或签证续签被拒等不利后果。

### 外商投资企业登记

须在产业通商资源部（经KOTRA）办理**外商投资企业登记**。通过该登记，可享受作为外商投资企业的各种税收优惠和政策支持。登记后签发的外商投资企业登记证也是D-8签证申请的必要文件。

### 四大社会保险

雇用员工时，须加入国民健康保险、国民年金、雇佣保险、产业灾害补偿保险等**四大社会保险**。法人设立后，在雇用员工时向管辖机构申报加入。外国人代表理事本人也须加入健康保险和国民年金，具体加入义务可能因居留期限而异。

### 正式开设法人银行账户

取得营业执照后，以法人名义开设**正式银行账户**。该账户与投资资金专用账户不同，用于日常经营资金的收支。开户时需提供营业执照、法人登记簿副本、代表理事身份证明等。

### 税务记账及申报准备

在韩国运营法人将产生**增值税**（按季申报）、**法人税**（按年申报）、**预扣税**（发放工资时）等多种税务申报义务。通常委托税务师或会计师负责记账和税务申报，从创业初期就与税务专业人士合作，是长远节约成本的有效方式。

### 外国人登记及地址申报

持D-8签证入境韩国后，须在**90天内办理外国人登记**。在管辖出入境管理局领取外国人登记证，该证件在韩国充当身份证明。变更居住地址时，须在14天内申报变更。

## 费用说明

以下是从法人设立到D-8签证取得的预估费用明细。实际费用可能因法人类型、资本金规模、营业场所位置等因素而有所不同。

| 费用项目 | 预估金额 | 备注 |
|-----------|-----------|------|
| 投资资金 | 1亿韩元以上 | 作为法人资本金，用于事业运营 |
| 公证费 | 约30万~80万韩元 | 公司章程、会议记录公证 |
| 登记许可税 + 教育税 | 约40万~120万韩元 | 因资本金规模而异 |
| 法务士手续费 | 约50万~100万韩元 | 法人设立登记代理 |
| 办公室租赁押金/月租 | 另行计算 | 因地区、规模差异较大 |
| 行政士/咨询服务费 | 约150万~700万韩元 | 按难度分级：简单150~300万、一般300~500万、复杂500~700万 |
| 上市公司进入韩国（含许可） | 另行协商 | KOSDAQ/KOSPI上市公司进入韩国，涉及行业许可、合资、并购等综合事项 |
| D-8签证手续费 | 约13万韩元 | 以变更居留资格为基准 |
| 税务师月记账费 | 约10万~30万韩元 | 每月固定支出 |
| 印章制作费 | 约3万~5万韩元 | 法人印章、使用印章 |

1亿韩元投资资金作为法人资本金用于事业运营，并非纯粹"消耗"的费用。但除投资资金外，法人设立相关附带费用预计约为**200万~500万韩元**。加上办公室租赁费用，初期定居成本可能相当可观，因此充分的资金规划十分重要。

## 所需时间

全流程所需时间取决于文件准备情况和相关机构的处理速度，一般基准如下：

| 阶段 | 所需时间 | 备注 |
|------|----------|------|
| 外国人投资申报 | 1~3天 | KOTRA或银行 |
| 投资资金汇款及到账确认 | 3~7天 | 含海外汇款时间 |
| 法人设立登记 | 3~7天 | 以文件齐全为基准 |
| 营业执照办理 | 1~3天 | 税务局处理 |
| 外商投资企业登记 | 3~5天 | 经KOTRA |
| D-8签证申请及签发 | 7~30天 | 因审查难度而异 |
| **合计** | **约3~6周** | **文件准备时间另计** |

以上时间以所有文件齐全为前提。文件准备可能额外需要1~2周，尤其是需从海外获取的文件（本国无犯罪记录证明、印鉴证明等）耗时可能更长。总体而言，从开始准备到获得D-8签证，预留**约1至2个月**较为现实。

委托专业行政士或法务士办理全流程，可以最大限度地缩短各环节的等待时间，通过同步处理可并行的程序来缩短总体所需时间。

## 常见问题 (FAQ)

**Q1：从法人设立到D-8签证签发总共需要多长时间？**

文件齐全的情况下约需3~6周。加上文件准备时间，建议预留1~2个月。委托专业人士代办可通过各环节并行处理缩短时间。

**Q2：一定要有办公室吗？可以用住宅地址登记吗？**

需要能够实际开展业务的营业场所地址。根据行业不同，有时也可以将住宅登记为营业场所，但D-8签证审查非常重视事业的真实性，因此建议使用独立办公室或共享办公空间。出入境管理局可能进行实地检查，因此该场所必须是实际进行工作活动的空间。

**Q3：持D-8签证可以邀请家人吗？**

可以。D-8签证持有者可以为配偶和未成年子女申请**F-3（随行）签证**。邀请家人时需提交经海牙认证（Apostille）或领事认证的亲属关系证明文件（结婚证明、出生证明等）。F-3签证的居留期限不得超过D-8签证持有者的居留期限。

**Q4：1亿韩元投资资金可以用于经营吗？**

可以。投资资金作为法人资本金，可**自由用于**合法的经营目的。可用于办公室租赁、员工工资、材料采购、市场营销等正当经营支出。但需注意，如设立后立即将投资资金全额提取用于个人用途，可能被认定为事业不具实质性。

**Q5：D-8签证可以变更为F-5永久居留权吗？**

可以。持D-8签证在韩国居留**5年以上**，并维持投资状态、雇用韩国国民等满足一定条件后，可申请变更为[F-5永久居留权](/visa/f5)。获得F-5永久居留权后，可在韩国不受居留期限限制地自由居住和从事经济活动。

**Q6：一人公司也能申请D-8签证吗？**

可以。股东仅1人的一人公司（一人股份有限公司）也被认定为外商投资法人，可申请D-8签证。但代表理事（投资者）须实际经营管理事业，且须能够确认事业的真实性。

**Q7：投资韩国已有法人也能获得D-8签证吗？**

可以。通过收购韩国现有法人10%以上（1亿韩元以上）的股份进行外国人投资，此种情况也可申请D-8签证。对现有法人的投资分为**股份收购**和**参与增资**两种方式，各自程序不同，建议咨询专业人士。
