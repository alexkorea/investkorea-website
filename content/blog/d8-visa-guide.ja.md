## はじめに

D-8投資ビザは、韓国で事業を設立・運営したい外国人にとって最も重要な在留資格のひとつです。半導体、バイオテクノロジー、電気自動車をはじめとするあらゆる分野のビジネスチャンスを見据えているなら、D-8ビザは直接投資と事業経営のための法的枠組みを提供してくれます。

本ガイドでは、韓国法人の設立から外国人登録まで、すべてのステップを詳しく解説します。

## D-8ビザの概要

D-8ビザは、外国人が韓国で**直接企業を設立・運営する**ことを可能にするビザです。アジアで最もダイナミックな市場のひとつに直接アクセスしながら、韓国の安定した事業環境の恩恵を受けることができます。

### 基本情報一覧

| 項目 | 詳細 |
|---|---|
| **ビザ種類** | D-8（法人投資） |
| **初回有効期間** | 1年（延長可能） |
| **最低投資額** | 1億ウォン（約1,000万円） |
| **申請対象** | 韓国法人に投資する外国人 |
| **家族ビザ** | 配偶者・未成年の子どもはF-3ビザ取得可能 |

## 法人設立の手続き

D-8ビザを申請する前に、まず韓国法人を登記する必要があります。一般的な流れは以下のとおりです：

1. **社名の決定** -- インターネット登記所で名称の利用可否を確認
2. **定款の作成** -- 会社の目的、資本構成、ガバナンスを明記
3. **仮口座の開設** -- 必要な資本金（最低1億ウォン）を払い込む
4. **管轄裁判所の登記所で法人登記** を完了
5. **国税庁で事業者登録証を取得**
6. **会計・税務システムを構築** -- 韓国の法規に準拠

<div style="background:#FEF3C7; border-left:4px solid #F59E0B; padding:12px 16px; border-radius:0 8px 8px 0; margin:16px 0; font-size:14px;">⚠️ <strong>注意：</strong> 資本金は韓国の銀行を通じて海外から送金する必要があります。外貨送金証明書はD-8ビザ申請の必須書類ですので、必ず保管してください。</div>

## ビザ申請の要件

D-8ビザの取得には、以下の条件を満たす必要があります：

- **法人登記済み**であり、最低資本金が払い込まれていること
- **事業計画書** -- 事業内容、予想売上、採用計画を詳述
- **韓国国内の事業所住所**（バーチャルオフィスは認められない場合があります）
- **株主および役員の情報** -- パスポートのコピーと経歴書類を含む
- **外国人投資届出** -- KOTRAまたは指定外国為替銀行を通じて完了

## 外国人登録

D-8ビザで入国後、**90日以内**に外国人登録を行う必要があります。手続きの流れは以下のとおりです：

1. パスポートとビザを持参して管轄の出入国管理事務所を訪問
2. 外国人登録申請書を提出
3. 証明写真を提出し、登録手数料を支払う
4. **外国人登録証（ARC）** を受領 -- 通常2～3週間で発行

外国人登録証の番号は韓国における身分証番号として機能し、銀行口座の開設、住居契約、携帯電話の契約など、韓国生活のあらゆる場面で必要になります。

## 申請の全体フロー

D-8ビザ申請の完全なタイムラインは以下のとおりです：

1. **韓国法人の設立**（2～4週間）
2. **指定銀行への外国人投資届出**
3. **必要書類の準備** -- 事業者登録証、事業計画書、賃貸契約書、資本金送金証明
4. **ビザ申請の提出** -- 出入国管理事務所（韓国国内）または韓国大使館（海外）
5. **審査・承認の待機**（通常3～6週間）
6. **韓国への入国**（海外申請の場合）および外国人登録の完了

<div style="background:#EFF6FF; border-left:4px solid #3B82F6; padding:12px 16px; border-radius:0 8px 8px 0; margin:16px 0; font-size:14px;">💡 <strong>ヒント：</strong> すでに別のビザ（C-3やD-2など）で韓国に滞在中の場合、出国せずにD-8への在留資格変更が可能な場合があります。出入国管理事務所にご確認ください。</div>

## ビザの更新

D-8ビザの初回有効期間は**1年**で、延長が可能です。更新の審査では、出入国管理局が以下の点を評価します：

- **事業の継続性** -- 会社は現在も活動しているか？
- **財務状況** -- 売上に上昇傾向があるか、または合理的なスタートアップ期の損失か？
- **雇用状況** -- 韓国人を雇用しているか？
- **税務コンプライアンス** -- 法人税の申告・納付は適切か？

### 更新に必要な書類

| 書類 | 目的 |
|---|---|
| 事業運営報告書 | 継続的な事業活動の証明 |
| 財務諸表（該当する場合は監査済み） | 財務状況の証明 |
| 納税証明書 | コンプライアンスの確認 |
| 更新版事業計画書 | 今後の方向性の提示 |
| 従業員給与記録 | 雇用創出の証明 |

## D-8ビザのメリット

D-8ビザの保有は、以下のような実際的なメリットをもたらします：

- 韓国の**1.7兆ドル規模の市場**に直接アクセス
- 外国投資企業向けの**政府支援プログラム**（補助金、助成金、インキュベーション）
- 配偶者・未成年の子どもへの**家族ビザ（F-3）** の付与
- 5年以上の継続投資による**F-5永住権への道**
- 登録された外国人投資家としての**銀行・金融サービス**へのアクセス

## 注意すべきポイント

D-8ビザを目指す際は、以下の点にご注意ください：

- **書類の正確性が極めて重要** -- 書類の不備や矛盾は、審査遅延の最大の原因です
- **実質的な事業活動を維持すること** -- 会社に実態がないと判断された場合、調査が入る可能性があります
- **記録を丁寧に保管すること** -- 更新時の財務透明性がビザ維持の鍵です
- **変更は速やかに届け出ること** -- 住所変更、事業内容の変更、役員の変更は出入国管理局への届出が必要です

<div style="background:#FEF3C7; border-left:4px solid #F59E0B; padding:12px 16px; border-radius:0 8px 8px 0; margin:16px 0; font-size:14px;">⚠️ <strong>注意：</strong> 実質的な事業活動を維持しない場合、ビザが取り消される可能性があります。D-8ビザは受動的な投資手段ではなく、積極的な経営参加が求められます。</div>

## 申請のコツ

- **認定行政士（ヘンジョンサ）に依頼する** -- 書類作成と届出を代行してもらえば、時間の節約とミスの軽減につながります
- **説得力のある事業計画書を準備する** -- 韓国経済への貢献（雇用創出、技術移転、輸出ポテンシャル）を明確に示すこと
- **チェックリストを作成する** -- すべての必要書類を一覧にし、提出前に一つずつ確認
- **早めに着手する** -- 法人登記だけでも2～4週間かかります

## よくある質問

**Q1. D-8ビザの有効期間はどれくらいですか？**

初回の有効期間は1年です。継続的な事業活動を証明することで、延長が可能です。

**Q2. D-8ビザの更新方法を教えてください。**

現在のビザが満了する前に、出入国管理事務所に事業運営報告書、財務諸表、納税証明書を提出してください。

**Q3. D-8ビザでどのような事業ができますか？**

直接投資、技術移転、研究開発関連の事業が対象です。小売、製造、サービス業も、投資要件を満たせば対象となります。

**Q4. 初回申請に必要な書類は何ですか？**

事業者登録証、外国人投資届出書、事業計画書、賃貸契約書、資本金送金証明、パスポートが主な書類です。

**Q5. 最低投資額はいくらですか？**

標準的な最低額は1億ウォン（約1,000万円）です。政府支援のスタートアッププログラムでは、より低い基準が適用される場合があります。

**Q6. 外国人登録は必ず必要ですか？**

はい。韓国入国後90日以内に、管轄の出入国管理事務所で登録する必要があります。

**Q7. 将来、事業を他国に移転できますか？**

可能ですが、韓国での事業を適切に整理し、出入国管理局に届け出ることで、将来のビザ申請に影響が出ないようにしてください。

**Q8. 申請時に面接はありますか？**

場合によっては、出入国管理局が事業と投資の正当性を確認するために面接を求めることがあります。初回申請者の場合に多い傾向があります。
